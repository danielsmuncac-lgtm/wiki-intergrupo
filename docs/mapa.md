# Mapa de Conocimiento Organizacional

A continuación se presenta el flujo interno del conocimiento en Intergrupo.

---

##  Ciclo del Conocimiento

1. **Dirección Estratégica:** Define lineamientos.  
2. **Gerencias:** Traducen estrategia en acciones.  
3. **Equipos Técnicos:** Crean y documentan conocimiento.  
4. **Clientes:** Retroalimentan y reinician el ciclo.
