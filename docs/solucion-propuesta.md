# Solución Tecnológica Propuesta

La propuesta se basa en optimizar los canales de gestión del conocimiento mediante herramientas modernas, escalables y alineadas con el sector TI.

---

##  1. Capa de Infraestructura
- Microsoft Azure  
- AWS EC2 para escalabilidad  
- GitHub Enterprise  

---

##  2. Capa de Aplicación
- Confluence / Wiki interna  
- Jira para gestión de proyectos  
- Integración con Teams  

---

##  3. Capa de Datos
- Azure SQL / PostgreSQL  
- Repositorios centralizados  
- Lakehouse para analítica
