# Canales de Comunicación y Conocimiento

Intergrupo utiliza una variedad de canales formales, informales y digitales para distribuir y gestionar el conocimiento.

---

##  Canales Formales
- Reuniones programadas  
- Correo corporativo  
- Intranet  
- Jira  
- Microsoft Teams  

---

##  Canales Informales
- Charlas en zonas comunes  
- Grupos internos de WhatsApp  
- Conversaciones espontáneas  

---

##  Canales Digitales
- GitHub  
- Repositorios de código  
- Bases de datos compartidas  
- Wikis internas
